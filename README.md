E-Commerce Order Management System
📌 Project Overview

The E-Commerce Order Management System is a Business Analyst case study designed to address common challenges in e-commerce delivery operations, including order tracking issues, payment visibility, and delivery coordination.

The project focuses on gathering business requirements, defining workflows, creating wireframes, and documenting user stories to design a Minimum Viable Product (MVP) for an order management platform.

🎯 Business Problem

Many e-commerce businesses face operational challenges such as:

Manual errors in order tracking
Lack of real-time delivery updates
Difficulty tracking pending payments
Poor coordination between sellers and delivery agents

These issues impact customer satisfaction and operational efficiency.

🎯 Project Objectives
Provide real-time order tracking
Improve payment visibility for sellers
Simplify delivery assignment process
Automate sales and performance reporting
Reduce operational delays and manual errors
